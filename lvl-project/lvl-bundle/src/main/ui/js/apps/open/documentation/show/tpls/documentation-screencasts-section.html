Overview and general usage

This demonstration recorded for the 1st review of the project EUBrazilCC presents an overview of the LeishVL portal. Although it was recorded with the early prototype, the information is still current.


Deploying new instances

A second instance of the LeishVL service is started from the EGI AppDB and configured using the IM. The new instance adds new processing capacities new to an existing deployment of the LeishVL .


Exporting items from a collection

The LeishVL allows users to select and export items from a collection to the supported file formats. In this example, a list of sequences are selected from the leishmania collection and exported to a FASTA file.
