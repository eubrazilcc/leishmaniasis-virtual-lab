Documentation

The following documentation correspond to the latest stable version (0.0.2) of the Leish VirtLab, released on March 2015.

Screencasts

Overview and general usage

This demonstration recorded for the 1st review of the project EUBrazilCC presents an overview of the Leish VirtLab portal. Although it was recorded with the early prototype, the information is still current.


Deploying new instances

A second instance of the LVL service is started from the EGI AppDB and configured using the IM. The new instance adds new processing capacities new to an existing deployment of the Leish VirtLab.
